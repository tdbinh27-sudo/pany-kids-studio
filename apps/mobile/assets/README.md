# Mobile Assets — Placeholders Required

Before first build, add these PNG files to this folder:

| File | Size | Purpose |
|------|------|---------|
| `icon.png` | 1024×1024 | App icon (iOS + Android) |
| `splash.png` | 1284×2778 | Splash screen on app launch |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon (foreground only, 25% padding) |
| `favicon.png` | 48×48 | Web favicon (when running `expo start --web`) |
